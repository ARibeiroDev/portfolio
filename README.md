# ARibeiroDev Portfolio

This is my **personal portfolio** showcasing my work and skills as a Full-Stack Web Developer. It is built with **React**, **TypeScript**, and **TailwindCSS**, featuring a clean, modern, and responsive design.

## Overview

The portfolio includes:

- **Hero Section**: Introduction, CTA buttons.
- **About Section**: Shows my frontend, backend, and tooling skills.
- **Projects Section**: Displays my projects with descriptions, tech stack, and live/source links.
- **Contact Section**: A working contact form with spam protection and email integration.
- **Animations & UX**: Smooth scroll animations, a custom loading screen, and responsive mobile navigation.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS
- **Animations & Effects:** ScrollReveal, custom loading/ping animations
- **Icons:** react-icons
- **Build Tool:** Vite

## Purpose

This portfolio serves as:

- A professional showcase of my skills and projects.
- A demonstration of clean, maintainable, and scalable frontend code.
- A way to provide potential employers or clients with an easy way to view my work and get in touch.
